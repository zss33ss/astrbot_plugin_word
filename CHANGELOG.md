# 更新日志

## v1.1.0 (2026-06-23)

### 新功能
- 新增 `max_total_chars` 配置项：单次回复总字数软限制（默认 500 字）
- 新增 `max_message_count` 配置项：单次回复消息条数上限（默认 3 条）

## v1.0.0 (2026-06-23)

### 新功能
- 首个正式版本发布
- 通过 `on_llm_request` 钩子在 LLM 请求前注入说话风格指令
- `style_prompt` 可自定义，支持后台实时编辑
- `enable` 开关，随时启用/禁用
- 防重复注入标记，兼容其他同样修改 system prompt 的插件
